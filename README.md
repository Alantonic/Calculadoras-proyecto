Esto es un proyecto propio de una pagina web, cuyo proposito es comprender herramientas basicas 
para la creación de paginas web funcionales e interactivas para el usuario, ademas de aplicar 
conocimiento aprendido a base de proyectos universtiario. Se piensa desarrollar dos calculadoras 
diferentes y funcionales, ademas de guardar proyectos diferentes en la misma pagina.

#Lenguajes utilizados

  **HTML** 
  Es una lenguaje de marcado que se utiliza para organizar contenido, fuera de lo que es de logica o
  diseño.

  **CSS**
  Es un lenguaje de diseño grafico, por lo que permite dar formato, color y una estructura 
  visual, esta basado en el tema de UX y UI

  **Javascript**
  Es un lenguaje orientado a objetos para el desarrollo de paginas web. Permite hacer paginas 
  dinamicas. En este caso se utilizo para agregar la logica en la calculadora simple.

  **PHP**
  Es un lenguaje de programación para desarrollar paginas web que permite gestionar datos.

  #Interfaces
  - Pagina principal: Los usuarios deben de ver un interfaz que muestré dos opciones de tipo de
    calculadora, ademas de haber un scrollbar que será utilizado para almacenar futuros proyectos.
  - Index: Se piensa hacer un area de registro y otro de iniciar sesión, creando una base de datos
    MySQL y PHP.
  - Calculadora simple: Es una calculadora funcional creado con HTML, CSS y Javascript, busca que
    los usuarios puedan hacer operaciones de forma sencilla y visual.
  - Calculadora Cientifica: En desarrollo.

  #Estructura del proyecto

  El proyecto está organizado en diferentes carpetas para separar responsabilidades:

   - CSS: contiene los estilos visuales
   - JS: lógica de la aplicación
   - PHP: scripts de prueba y backend
   - Imagenes: recursos gráficos
   - HTML: páginas principales del sistema

Esta estructura facilita el mantenimiento y escalabilidad del proyecto.
